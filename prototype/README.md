# sipdump.pl

Extracts SIP traffic packets from a given PCAP file

## requirements

This script requires following modules to be installed from CPAN:

* Net::Pcap
* Net::Frame
* Net::Packet::Layer7


# pcaputil.pl

Generates gnuplot definition file from a PCAP file to draw BPS and PPS graphs

## requirements

This script requires following modules to be installed from CPAN:

* Net::Pcap
